# Data Analysis Mentoring Repository

데이터 분석 멘토링 과정에서 사용한 강의 자료 및 실습 코드를 정리한 저장소입니다.  
Pandas와 Matplotlib를 중심으로 데이터 전처리 및 시각화 기초를 학습하고,  
실제 데이터를 활용한 예제 및 실습을 진행했습니다.

- 실제 공공데이터를 활용한 데이터 분석 실습
